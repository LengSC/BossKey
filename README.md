简介
---

本软件研发的目的就是为了VAN！是作者寒假无聊搞出来的。但是，如果你是一个一个摸鱼小行家，不妨逝一世它！

> 向往自由但是被资本🥔压榨得抬不起头？
>
> 想要摸鱼但老板装了114514K立方氮化硼狗眼？
>
> 以及各种乱七八糟的原因？

如果你被以上问题所困扰，本软件或许是你想要的。

这是一个可以**快捷隐藏应用窗口**的软件。也就是说，你在摸鱼时，如果老板突然诈出，你可以迅速装作在认真工作，因此你因**摸鱼致死**的可能性会**大大减小**。

功能 & 特色
---

* 隐藏或显示绑定的摸鱼窗口。
* 在隐藏摸鱼窗口的同时，可以显示（或置顶）一个关于工作的窗口。
* 可自定义快捷键，更加适应使用环境。
* 软件小巧，资源占用可忽略。
* 莫得广告。
* 莫得收费。
